Healthcare Symptom Checker

AI-Powered Medical Symptom Analysis System

Using Groq LLM & Flask Backend

Deeper Dive into the Problem Statement

  • Online health searches often provide misleading or unstructured information
  • Patients struggle to understand which symptoms require immediate attention
  • Lack of reliable first-level triage before doctor consultation
  • Need for responsible AI-driven educational health guidance
  • Traditional symptom checkers lack context and personalization

Solution Structure

  • Users input symptoms through intuitive web interface
  • AI analyzes and suggests possible conditions ranked by likelihood
  • Provides actionable next steps and care recommendations
  • Educational disclaimers included in all responses
  • Query history tracking for reference and analysis
